1.接口配置文件写入Excel 中，共有 用例编号,接口名称,基本URL,请求路径,请求参数,参数类型(dict,json),请求类型(GET,POST,PUT),期望结果包含,功能函数 9列

2.功能函数，当配置表格不满足需求时，可以针对个别接口在function.py文件中自定义方法，在功能函数列填写对应方法名即可

3.根据Excel模板用例生成不同的测试界面，可参考已写好的接口文件
------------------------------------------------------------------------------------------
4.function.py 文件用于重写接口方法。方法名采用excel文件中配置的名称，方法最后返回响应结果response即可.

5.function.py 文件需要与可执行文件放在同一层目录下

-------------------------------------------------------------------------
5.运行结果放在当前目录下的result目录下

-----------------------------------
6.电脑需要有python环境
